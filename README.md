The Rock Paper Scissors Game is a simple and fun computer-based game where a user
plays against the computer. In this game, the player chooses either Rock, Paper, or Scissors,
and the computer also randomly selects one option. The winner is decided based on the game
rules

Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock

This project is designed to improve programming logic, conditional statements, loops, and
random number generation skills. It is a beginner-friendly project and can be developed using
Java.
